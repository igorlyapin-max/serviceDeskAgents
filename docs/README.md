## Repository Documentation Rules

- `docs/PLAN.md` and `docs/STAGE*.md` are local working notes and are not committed to git.
- Permanent user-facing documentation is maintained in Russian and English.
- Technical identifiers are not translated: API paths, environment variables, JSON fields, service names, Kafka topics, enum/status ids, tool names, endpoint ids and operation ids.
- The debug console is documented as `/debug`; `/operator` is only a compatibility alias.
