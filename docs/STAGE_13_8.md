# Этап 13.8: выбор условий и пакета передачи

Этап 13.8 делает блок `5. Решение и эскалация` управляемым через предметные поля, а не через технические CSV-строки.

## Реализовано

- В `escalation_policies` используются новые поля:
  - `handoff_conditions`;
  - `handoff_package`.
- Старые поля `l2_conditions` и `escalation_package` удалены из активной модели и не поддерживаются.
- В Admin UI блока `5`:
  - условия передачи выбираются чекбоксами;
  - пакет передачи выбирается чекбоксами;
  - обязательные пункты `Собранные слоты` и `Текст уведомления пользователя` нельзя отключить;
  - у каждого пункта есть короткое пояснение.
- Operator UI показывает выбранные условия и состав пакета передачи.

## Валидация

- `handoff_conditions` должен быть непустым.
- `handoff_package` должен быть непустым.
- `handoff_package` обязан содержать:
  - `slots`;
  - `user_notification`.
- Major Incident threshold должен быть не меньше 10.
- Payload со старыми полями `l2_conditions` или `escalation_package` не проходит JSON Schema.
